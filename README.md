**Небольшой скрипт для конвертации xls файлов в xml для загрузки в ГМУ отчетности**

- Бюджетная смета
- Целевые средства
- Планы ФХД
- Отчет о результатх деятельности

файлы для конвертации ложим в каталог /in

конечные xml падают в /out

конечно xls Должны быть в определенном формате!

Отчет о результате деятельности работает но не все разделы
многое еще необходимо дорабатывать!

необходимо указать параметры для конвертации:

- --fhd для конвертации планов ФХД
- --bs для конвертации Бюджетные средства
- --rd для конвертации Результаты деятельности
- --cs для конвертации Целевые средства

пример:

go run main.go --fhd
